URL 확장 API

하나의 URL로 상태 코드, 헤더, 리다이렉트 체인, 최종 주소까지 한 번에 가져오는 인스펙터 API. 네트워크 디버깅, 링크 검증, 미리보기용으로 쓸 수 있어요.

실시간 테스트

대기 중 
요청을 보내면 이곳에 JSON 응답이 표시됩니다.

API 스펙

엔드포인트

GET https://exurl-api.dxdxffg.com/api/exurl?url=<ENCODED_URL>

쿼리 파라미터

  • url (필수): 검사할 대상 URL
  • follow (선택): 1이면 리다이렉트 체인 추적
  • head (선택): 1이면 HEAD 기반 검사

curl 예시

curl https://exurl-api.dxdxffg.com/api/exurl?url=https://example.com/"

JavaScript 예시

async function inspectUrl(url) {
  const endpoint = "/api/exurl?url=" + encodeURIComponent(url);
  const res = await fetch(endpoint);
  if (!res.ok) throw new Error("HTTP " + res.status);
  const data = await res.json();
  console.log(data);
  return data;
}

주의사항

  • 이 API 자체는 서버/워커에서 실행되므로, 클라이언트의 CORS 제약을 우회할 수 있습니다.
  • 프록시 역할을 하기 때문에 민감한 내부망 주소를 그대로 넘기지 않도록 주의하세요.
  • 폭주 방지를 위해 도메인 화이트리스트, 레이트 리밋 등을 적용하는 걸 권장합니다.
